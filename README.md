# 🚀 GOD-TIER AUTOBUY STACK

## 📋 Overview
ระบบ AI อัจฉริยะด้าน AUTOBUY อัตโนมัติทุกเว็บ ที่เชื่อมต่อและ Orchestrate ระบบ GOD-TIER AUTOBUY STACK

## 🏗️ Architecture Components

### 1. **CURSOR**
- GUI Layer, รับ/ส่ง Prompt, เชื่อมผู้ใช้กับ AI, และเป็นจุด Prompt Engineering

### 2. **LangGraph**
- Task Engine สำหรับวางแผนและสั่งงาน (Task Planner/Executor)

### 3. **Agent-S2**
- ควบคุม GUI จริงในเครื่อง เช่น คลิก/พิมพ์/ลากวาง อัตโนมัติ

### 4. **OpenInterpreter**
- ควบคุม Shell/CLI จริง เช่น สั่ง Python, Bash, System Automation

## 🛠️ Installation & Setup
```bash
# Install dependencies
pip install -r requirements.txt

# Initialize the stack
python src/core/orchestrator.py --init

# Start the system
python src/core/orchestrator.py --start
```
